# PacketEye 👁️

**PacketEye** is an advanced WiFi network monitoring tool built with Python.  
It scans devices connected to your local network and displays useful information such as IP address, MAC address, device name, vendor, and activity status.

---

## 🚀 Features

- 🔎 Scan devices connected to your WiFi network
- 📡 Detect IP and MAC address
- 🧠 Hostname detection
- 🏢 MAC vendor lookup
- 📊 Device activity status
- 🌐 Live network map
- 🔄 Automatic refresh monitoring
- 🎨 Terminal UI using Rich library

---

## 📦 Requirements

- Python 3.8+
- Linux / Windows / macOS
- pip
